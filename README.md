# Neovim config

Prøver meg på en neovim-konfigurasjon. Denne er basert på [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim).

## Oppsett på macOS med Neovide

1. Installer Neovide via Homebrew:
```
brew install --cask neovide
```

2. Gå til mappen `~/.config/nvim` og kjør:
```
git clone git@github.com:SimenHolmestad/nvim.git
```

3. Start Neovide via Spotlight (`cmd+space neovide`)
